1. INTRODUCTION

The Codec Component contains a set of Java classes that provide encode/decode functionality and helper methods for various data formats. The following classes are included:

  • Binary Encoders/Decoders
    • Base64 - Encoding/decoding of the popular base64 format.
  • Phonetic Encoders
    • Soundex - Encoding of Strings to soundex values.
    • Metaphone - Encoding of Strings to metaphone values, a more advanced soundex-like algorithm.
    • RefinedSoundex
